idea创建springboot项目

第一步：

打开idea 选择 file > new > project

选择spring initializr 点击 next

填写完项目必要的信息后点击 next

勾选 springweb 点击next

点击finish

提示当前窗口或新窗口打开时，自己随意选。

完成后展开目录结构如下：

此时项目框架已经完成

点击右上角的 运行按钮就可以启动了

下方的日志中即可看到启动成功！
